# 需求文档 — 历史粘贴板

## 项目概述

一个运行在 macOS 上的历史粘贴板软件，自动记录用户复制的文字和图片，支持查看、搜索、粘贴历史记录。

## 功能需求

### F1 — 剪贴板监听
- 自动监听系统剪贴板变化
- 支持记录**文字**和**图片**两种内容
- 去重：相同内容不重复记录

### F2 — 历史存储
- 数据持久化存储在本地（`~/Library/Application Support/HistoryClipboard/`）
- 元数据：JSON 格式
- 图片：PNG 文件
- 超过保留时长的条目自动清理

### F3 — 保留时长设置
- 可选：1 天 / 3 天 / 5 天
- 超过时长的非置顶条目自动删除
- 默认值：3 天

### F4 — 内容展示
- 按**时间降序**排列（最新在上）
- 置顶条目始终排在顶部
- 卡片形式展示每条记录
- 文字预览（超过 150 字截断）
- 图片以缩略图展示

### F5 — 置顶功能
- 可手动置顶 / 取消置顶
- 置顶条目**永不过期**
- 置顶条目有一条蓝色 "置顶" 标签

### F6 — 删除功能
- 可手动删除任意记录
- 删除前需确认
- 删除图片时同时删除对应文件

### F7 — 搜索功能
- 支持关键词搜索（仅搜索文字记录）
- 实时过滤
- 不区分大小写

### F8 — 粘贴功能
- 点击记录 → 写入系统剪贴板 → 模拟 Cmd+V 粘贴
- 适用文字和图片

### F9 — 软件形态
- **菜单栏图标**：点击弹出面板，显示最近 5 条
- **独立主窗口**：完整历史列表、搜索、管理
- **设置窗口**：保留时长、开机自启

### F10 — 开机自启动
- 用户登录时自动运行
- 可在设置中开关
- 使用 SMAppService API

## 非功能需求

| 需求 | 说明 |
|------|------|
| 性能 | 剪贴板轮询间隔 0.5s，不影响系统性能 |
| 可靠性 | 数据实时写入磁盘，不因崩溃丢失 |
| 兼容性 | macOS 14.0 (Sonoma) 及以上 |
| UI 风格 | 简洁直观，淡蓝色主色调 |

## 约束条件

- 不进入 App Store（无需沙盒）
- 需要辅助功能权限用于模拟粘贴
- 仅支持 macOS 平台
